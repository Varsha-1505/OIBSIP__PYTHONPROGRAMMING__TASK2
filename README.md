BMI Calculator (Python)

A simple Body Mass Index (BMI) Calculator built using Python.
This program allows users to enter their weight (kg) and height (meters) to calculate their BMI and determine their weight category.

Features
Calculates BMI using the standard formula
Categorizes BMI into:
Underweight
Normal weight
Overweight
Obese
Handles invalid inputs using exception handling
Validates that height and weight are positive numbers
Allows users to perform multiple calculations in one run

BMI Formula
BMI is calculated using the formula:
BMI = weight (kg) / height² (m²)
